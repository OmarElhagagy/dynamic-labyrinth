Welcome to Secure-X

About Secure-X

Secure-X is a comprehensive Security Operations Center (SOC) dashboard providing real-time monitoring and analysis of threats, user activity, and system metrics.

Getting Started

  • Select a user from the dropdown or quick-select cards
  • Enter the corresponding password
  • Or create a new account using the "Create Account" tab
  • Use the theme toggle to switch between light and dark mode

Dashboard Sections

  • Dashboard: Live metrics, attack charts, and recent logs
  • Security: Firewall status, active threats, blocked IPs
  • Logs: Full event log with filtering by level and source
  • Users: Manage accounts and add new users
  • Settings: Notifications, security, and appearance options
